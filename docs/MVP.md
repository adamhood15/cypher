# Waterpark Mobile App MVP

## Vision

Build a modern Progressive Web App (PWA) for guests visiting our waterparks.

The application should improve the guest experience before, during, and after their visit while reducing friction for admissions, ticket management, event discovery, and communication.

The MVP should focus on the minimum features needed to launch to real users.

---

# Goals

Primary Goals

- Improve guest experience
- Increase season pass engagement
- Increase in-park spending
- Reduce customer service questions
- Create a direct communication channel with guests
- Promote flash sales, and special events

---

# Target Users

## Guest

A visitor purchasing daily tickets.

Needs:

- Buy tickets
- View tickets
- Find attractions
- View events
- Park Notifications
- Order food and beverage
- View Cabanas
- Book cabanas
- Event Calendar

---

## Season Pass Holder

Frequent visitor.

Needs:

- Digital season pass
- Exclusive offers
- Event calendar
- Park notifications
