# Design System

## Overall Feel

The app should feel like a clean, modern, amusement park app that prioritizes simplicity and ease of use.

Keywords:

- Fun
- Modern
- Bright
- Premium
- Fast
- Family-friendly
- Minimal
- Clean

Avoid:

- Clutter
- Gradients
- Cartoon styling
- Large blocks of text
- Outdated UI patterns

---

# Inspiration

Primary inspiration:

- Disney World App
- Universal Orlando App
- Airbnb
- Apple Wallet
- Uber
- Spotify

---

# Design Philosophy

The interface should prioritize:

1. Large touch targets
2. Fast navigation
3. Clear typography
4. Visual hierarchy
5. Plenty of whitespace

Users should never feel overwhelmed.

# Border Radius

Large rounded corners.

Cards should feel soft.

Buttons should be rounded.

---

# Shadows

Very subtle.

Avoid dramatic drop shadows.

---

# Cards

Every feature should live inside cards.

Examples:

Today's Hours

Upcoming Event

Season Pass

Weather

Notifications

See cards.png and cards-2.png for reference

---

# Icons

Use Lucide React.

Avoid emoji.

Use outlined icons whenever possible.

---

# Images

Use photography whenever possible.

Avoid illustrations.

Large hero images.

Rounded corners.

Use images in the references folder, read image names for context

---

# Animations

Keep animations subtle.

Use:

- fade
- slide
- scale

Avoid:

- bounce
- spin
- flashy transitions

---

# Mobile First

Design for iPhone first.

Android Second.

Tablet third.

Desktop last.

---

# Components

Create reusable components for:

- Buttons
- Cards
- Page Headers
- Navigation
- Alerts
- Empty States
- Loading States
- QR Cards
- Event Cards
- Ride Cards
